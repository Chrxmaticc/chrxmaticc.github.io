FAQ

How do I add Chrxmaticc AI to my server?

Just click the invite button on the home page or use /invite in any server that has the bot. You need Manage Server permission to add bots.

Is Chrxmaticc AI free?

Yes. 100% free. No premium, no paywalls, no "Pro" tier. Just pure chaos at no cost.

What AI model does it use?

We use Groq with multiple models — Llama 3.3 70B, GPT-OSS 120B, Qwen 32B, and more. You can switch between them anytime with /model.

Does it store my messages?

Only what you explicitly set with /personalinfo. No spying, no selling data, no training on your conversations. Privacy is real here.

Can I customize how the bot looks in my server?

Yes. Use /customize to change the bot's avatar, banner, bio, and nickname per server. Each server gets its own look.

What are AI modes?

10 different personalities — Unfiltered, Insane, Comedian, Silent, Paranoid, Depressed, Disappointed, Teacher, Vibe, and Suspicious. Switch with /mode.

How do I make the bot respond to me?

Use /chat to start a conversation, or just @mention the bot with your message. Server admins can set wake-up mode to make it respond automatically.

Is there a way to create custom commands?

Yes. Admins can use /customcmd create to make server-specific commands. Anyone in the server can trigger them by typing the command name.

How often does the bot update?

Constantly. Sometimes daily, sometimes weekly. We actually update unlike some bots that still run GPT 3.5 in 2026. You know who you are.

What if the bot is offline?

It's probably just a quick restart for an update. The bot auto-recovers. If it's down for more than a few minutes, something's wrong — but that rarely happens.
